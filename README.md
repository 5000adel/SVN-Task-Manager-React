# SVN Task Manager

A construction task management app built with React.

## Features
- Role-based login (Admin, Worker, Supervisor)
- Kanban style task board
- Activity logs

## Tech Stack
- React + Vite
- CSS Modules

## Getting Started Using Terminal

1. Clone the repo
   git clone https://github.com/5000adel/SVN-Task-Manager-React.git

2. Install dependencies
   npm install

3. Run the app
  cd svn-app
  npm run dev
